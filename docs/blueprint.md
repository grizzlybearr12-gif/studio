# **App Name**: CaterEase Connect

## Core Features:

- Catering Service Provider Listing: Display catering service providers with contact number, specialty, price per plate, rating, and reviews for each city in India.
- User Authentication: Allow users to log in using Google email or phone number with OTP verification.
- Service Provider Registration: Provide an interface for catering service providers to register their company with all necessary information.
- Search and Filter: Enable users to search for catering services by city, specialty, and price range.
- Rating and Reviews: Allow users to rate and review catering services.

## Style Guidelines:

- Primary color: A subdued, sophisticated gold (#BDB189), suggestive of catered events and fine dining. This choice also lends the app a high-end aesthetic.
- Background color: A dark, warm gray (#333330) for a luxurious feel that allows the content to stand out.
- Accent color: A muted olive green (#A2B59F), complementing the gold and enhancing the feeling of upscale dining.
- Font pairing: 'Playfair' (serif) for headlines, providing a classy, high-end feel, paired with 'PT Sans' (sans-serif) for body text to ensure readability.
- Use elegant, minimalist icons representing different catering specialties and functionalities.
- Employ a grid-based layout to present catering options in a clean and organized manner.
- Use subtle transitions and animations to enhance user engagement.
